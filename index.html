How it works

We configure it. You power it on.

Every kit ships pre-configured and ready — no app, no admin login, no IT visit to set it up.

01 — BUILT BEFORE IT SHIPS

We pair every node before it leaves us

One node ships as primary, the rest as extenders, all paired and locked to your kit's own network. There's nothing to configure when it arrives.

02 — CONNECT TO NETWORK

The primary node plugs into your gateway

It works with whatever's already on site — fiber, cellular, satellite, or a hardline. That's the only wired connection in the whole setup.

03 — PLACE WHERE SIGNAL DROPS

Set nodes wherever you have dead spots

Drop a node anywhere coverage gets weak — a deep corner, a stairwell, a lower floor. Power it on — battery or plugged in — and it joins the network automatically.

04 — BACK ONLINE

Crew connects like normal wifi

No special app, login, or device setup — your team gets back to project software, plans, comms, and cameras wherever a node reaches.
